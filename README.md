# Telegram Chess Arena

Telegram Web App для шахмат: онлайн-матчи, игра с компьютером, рейтинг и простой стильный интерфейс.

## Быстрый запуск в Telegram

1. Создайте бота через `@BotFather` и получите `BOT_TOKEN`.
2. Поднимите публичный HTTPS-адрес для приложения. Для теста подойдет ngrok:

```bash
ngrok http 3000
```

3. Создайте `.env`:

```env
BOT_TOKEN=токен_бота
PUBLIC_URL=https://ваш-url.ngrok-free.app
PORT=3000
DATABASE_PATH=./data/chess.json
```

4. Настройте кнопку Telegram автоматически:

```bash
npm run telegram:setup
```

5. Запустите сервер:

```bash
npm run dev
```

После этого откройте своего бота в Telegram и нажмите кнопку `Играть`.

## Локальный тест

Без Telegram можно открыть:

```text
http://localhost:3000/?devUser=demo
```

## Что уже есть

- Telegram-бот с кнопкой запуска Web App.
- Автоматическая настройка меню бота через `npm run telegram:setup`.
- Онлайн-очередь и партии через WebSocket.
- Игра против компьютера с быстрым эвристическим ходом.
- Рейтинг Elo после завершения онлайн-партий.
- JSON-хранилище пользователей и завершенных партий.
- Адаптивный интерфейс с доской, списком ходов и таблицей лидеров.
